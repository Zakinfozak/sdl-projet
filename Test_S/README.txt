STEP TO DISPLAY WINDOW 

1) go into /build
2) enter the command  : cmake .. (it will store all generated file into the build folder)
3) enter the commmad : make
4) the executable has been generated ( if no error occurs)
5) run the cmd : ./SDL_TEST/SDL_TEST ( running it directly from SDL_TEST repo might fail loading the image)
6) the window is displayed for few seconds

===> check SDL_TEST/Main.cpp for the code
